# Multi-Modal Demand Forecasting for Fast Fashion Products

One-page project showcase for the Istanbul Technical University Artificial Intelligence and Data Engineering senior design project.

## Website

https://tevhidenur-serdar.github.io/fashion-demand-forecasting-showcase/

## Project

The project combines product images, semantic captions, structured attributes, store and temporal context, and internal historical trends to forecast seven-week demand for newly released fashion products.
